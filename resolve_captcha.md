# Consideraciones previas
- Usar movimientos lentos y semierráticos, como si fueras un humano
- Imitar la cadencia humana de escritura, variando entre 50 y 80 palabras por minuto

# Set de instrucciones
- Entrar a https://es.sporcle.com/games/WillieG/crazy-captchas
- Clickar sobre el boton con el siguiente código HTML: 
``` HTML
<button title="Aceptar" aria-label="Aceptar" class="message-component message-button no-children focusable buttons-row accept-button sp_choice_type_11 last-focusable-el" style="opacity: 1; padding: 10px 15px; margin: 10px; border-width: 1px; border-color: rgb(255, 153, 0); border-radius: 5px; border-style: solid; font-size: 14px; font-weight: 700; color: rgb(255, 255, 255); font-family: arial, helvetica, sans-serif; background: rgb(255, 153, 0); width: calc(33% - 50px);">Aceptar</button>
 ```
- Clickar sobre el botón naranja "JUGAR QUIZ"
+ Repetir las siguientes instrucciones hasta finalizar los 30 captchas:
    - Scrollear ligeramente si es necesario para visualizar el captcha. Debes seguir scrolleando arriba o abajo hasta que seas capaz de ver todo el captcha correctamente, sin cortes.
    - Leer lo que se indique en la imagen del div con id="imgbox", es decir, interpretar que hay que escribir como captcha
    - Escribir exactamente lo que se ha interpretado que es el captcha
    - Repetir hasta que se completen los 30 desafíos.


# Test de éxito
- Si te detecta como un bot has fallado, es decir, no puedes fallar ningun captcha.
